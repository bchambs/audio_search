What is this?
audiosearch is an open source web application created in the Django web framework. This website acts as an interactive implementation of The Echo Nest's music services. It uses a python wrapper library (pyechnoest) to abstract the restful service calls into python objects. I created this project for two reasons:

1. I wanted to improve my front-end development.

2. I want to host an open project which can be criticized for its visual design as well as its source code structure.
